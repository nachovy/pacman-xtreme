# pacman-xtreme
A simple Web Pacman Game based on Java & Maven.

# Run Locally

    java -jar target/pacman-final-team-xtreme-1.0-SNAPSHOT-jar-with-dependencies.jar

Then, visit http://localhost:4567/.
